# HD Piano Landing Page

A responsive and modern landing page designed around an interactive piano-learning theme. The layout includes sliders, call-to-action sections, and a clean, user-friendly interface. Built with **HTML**, **CSS**, **Bootstrap**, and **Slick Carousel**, the page is fully mobile-friendly and easy to customize.
